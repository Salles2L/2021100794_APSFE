Este projeto consiste em uma aplicação de lista de tarefas (Todo List) dividida em uma arquitetura Cliente-Servidor.

📁 Estrutura do Projeto
/client: Contém os arquivos do frontend (HTML, CSS e JavaScript).

/server: Contém a API desenvolvida em Node.js e a lógica do backend.

🚀 Como executar a aplicação
1. Pré-requisitos
Certifique-se de ter o Node.js instalado em sua máquina.

2. Configurando o Servidor (Backend)
O servidor gerencia os dados e a lógica da aplicação.

Abra o terminal e navegue até a pasta do servidor:

Bash
cd server
Instale as dependências necessárias (como Express e CORS):

Bash
npm install
Inicie o servidor:

Bash
node main.js
O terminal deverá exibir: Todo List API rodando em http://localhost:8000.

3. Configurando o Cliente (Frontend)
Navegue até a pasta client.

Localize o arquivo index.html.

Abra este arquivo em seu navegador de preferência.

Dica: Se estiver usando o VS Code, você pode clicar com o botão direito no index.html e selecionar "Open with Live Server" para uma melhor experiência.

🛠️ Tecnologias Utilizadas
Frontend: HTML5, CSS3 e JavaScript Vanilla.

Backend: Node.js com ambiente de execução Express.
